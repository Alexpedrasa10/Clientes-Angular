Tutorial para crear este proyecto paso a paso.

1. Crear aplicación

Utilizando CLI, crearemos un nuevo proyecto con el comando: 

                    
                        $ ng new clientes-app

Luego, accederemos a la carpeta creada y crearemos un servidor para empezar con el desarrollo de nuestra aplicación. 

                    
                        $ ng cd clientes app
                        $ ng serve

2. Crear modulo

Con Angular CLI crearemos un modulo con el siguiente comando: 

                    
                        $ ng generate module clientes

Y lo imporatamos en el módulo principal 'app.module.ts'

                    
                       {{importModule}}
                    
                
                    
                        {{importModuleinApp}}

En este proyecto, utilizaremos ngModel, por lo que vamos a importar 'FormsModule', y también otras clases para poder validar nuestro formulario. 

                    
                        {{importFormsModule}}

Y lo declararemos en 'imports'.

                    
                        {{importsModule}}

3. Crear interface

Con esto definiremos el modelo de datos. No hay generador de este tipo de modelo, asique vamos a crearlo de forma manual. El archivo se va a llamar: cliente.model.ts
En este archivo coloco las interfaces de TypeScript que definen los datos de mi aplicación. 

                    
                        {{interfaceCliente}}

4. Crear servicio

Con Angular CLI crearemos un servicio usando: 

                    
                        $ ng generate service clientes

Luego, importaremos la interfaz creada en el paso anterior.

                    
                        {{importInterface}}

Codigo completo del servicio:

                    
                        {{codigoService}}

1. La propiedad "cliente" contiene los datos que va a mantener y la declaramos como privada para que no puedan ser tocadas directamente, solamente usando los métodos del servicio.

2. Agregar un cliente es un "push" al array de clientes, de un cliente recibido por parámetros.

3. Crear un nuevo cliente es simplemente devolver un objeto que debe respetar la interfaz, ya que en la función se especifica que el objeto es de tipo Cliente.

5. Importar el servicio en el módulo

Vamos a dirigirnos al módulo creado en el paso 2 (cliente.module.ts) y declarar el servicio.

                    
                        {{importService}}

Y declararlo en el array providers.

                    
                        {{importServiceProvider}}

6. Crear componente Clientes

Crearemos el componente con CLI, en él estará nuestro formulario y el listado de clientes.

                    
                        $ ng generate component clientes

También los importaremos en el módulo que hemos creado y delcararlo en 'declarations' y 'exports'

                    
                        {{componentModuleImport}}
                    
                
                    
                        {{componentModuleDeclaration}}

Ahora iremos al 'app.component.html' y pondremos el tag del component

                    
                        {{appClientes}}

7. TypeScript del componente

Comenzaremos importando el servicio y el modelo de datos. 

                    
                        {{importServideAndInterface}}

También importaremos algunas clases de '@angular/forms' que nos ayudará a validar el formulario que vamos a crear. 

                    
                        {{importFormsComponent}}

Código completo del TypeScript del componente: 

                    
                        {{codigoTypescriptComponent}}

1. El componente declara 3 propiedades: el cliente, el array de todos los clientes y la propiedad 'form' proveniente de Angular Forms.

2.En el constructor, que se ejecuta lo primero, conseguimos una instancia del servicio de clientes y de FormBuilder, mediante la inyección de dependencias. Luego, declararemos los tipos de datos requeridos del formulario para que este sea válido.

3. En el ngOnInit(), usaremos la función 'newCliente()' para la propiedad 'cliente' para que los campos que usemos esten vacíos al cargar la página, y también utilizaremos 'getClientes()' para el array de clientes para que al mandar el formulario, obtenga la lista de clientes.

8. Template del componente

Antes de pasar al template, instalaremos Bootstrap.

                    
                        $ npm install bootstrap jquery @popperjs/core

Luego de eso nos dirigimos al archivo “angular.json” y colocamos las siguientes instrucciones en los objetos “styles” y “scripts” en donde llamaremos a las propiedades css del bootstrap y las dependencias scripts correspondientes a las interacciones de cada uno. 

                    
                        {{bootstrapjson}}

Una vez instalado, pasaremos al HTML del componente:
